# HydraLearn

HydraLearn es una herramienta educativa de código abierto diseñada para facilitar el aprendizaje y la comprensión de conceptos fundamentales de redes.

## Estado del Proyecto

  * **Versión:** alpha\_0.0.1
  * **Estado:** Fase temprana de desarrollo.

## Requisitos Técnicos

Este proyecto ha sido desarrollado utilizando **Godot Engine 4.6**. Para ejecutar o colaborar en el proyecto, es necesario tener instalada esta versión específica del motor.

## Descargar ejecutables

El proyecto es compatible con entornos Windows y Linux.

1. Accede a la sección **Releases** del repositorio.
2. Selecciona la versión más reciente disponible.
3. Descarga el archivo correspondiente a tu sistema operativo:
   - Windows
   - Linux
5. Ejecuta el binario del proyecto.


## Instalación (Desarrollo)

1.  Clonar este repositorio en su equipo local.
2.  Abrir Godot 4.6.
3.  Importar el proyecto seleccionando la carpeta raíz del repositorio.
4.  Asegurarse de que todos los plugins incluidos en el repositorio estén habilitados en la configuración del proyecto.

## Contribuciones

Se aceptan contribuciones para mejorar y expandir HydraLearn. Para colaborar:

1.  Realice un fork del repositorio.
2.  Cree una nueva rama (branch) para sus cambios o nuevas funcionalidades.
3.  Utilice los plugins integrados en el repositorio para mantener la compatibilidad.
4.  Envíe un Pull Request con una descripción detallada de los cambios realizados para su revisión.
